📋 About

This page allows you to configure the upstream Anthropic API destination for the proxy running on port 8888. By default, the proxy forwards requests to https://api.anthropic.com, but you can override this to point to a different Anthropic-compatible endpoint (e.g., a local mock server, alternative provider, or custom gateway).

🔧 Usage

To use the Coding Agent Explorer proxy with Claude Code, set the ANTHROPIC_BASE_URL environment variable: 

export ANTHROPIC_BASE_URL=http://localhost:8888

Then run Claude Code normally. All API requests will flow through the proxy on port 8888 and be forwarded to the destination configured above.

⚙️ Configuration Sources

The Anthropic destination URL can be configured via:

  1. This web page (runtime configuration, highest priority)
  2. Environment variable: CODING_AGENT_EXPLORER_UPSTREAM_URL
  3. Configuration file: appsettings.json
  4. Default fallback: https://api.anthropic.com
