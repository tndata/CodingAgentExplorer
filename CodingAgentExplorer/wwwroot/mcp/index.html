🔌 Registering the MCP Proxy in Claude Code

The MCP Observer sits between Claude Code and the real MCP server. Set the destination URL above, then register the proxy as an MCP server in Claude Code using this command:

claude mcp add --transport http mcp_proxy http://localhost:9999

Claude Code will send all MCP traffic to http://localhost:9999, which this proxy forwards to the destination URL you configured above.

To remove the proxy registration later:

claude mcp remove mcp_proxy
